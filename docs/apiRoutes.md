| **Method** | **API**                      | **Auth** | **Description**                                                 
| ---------- | ---------------------------  |:--------:| ----------------------------------------------------------------
| POST       | /v1/login                    |   No     | Creates session token [login]
| POST       | /v1/users                    |   No     | Creates a new user [signup]
| GET        | /v1/users/me                 |   Yes    | Returns user info
| PATCH      | /v1/users/me                 |   Yes    | Updates user info
